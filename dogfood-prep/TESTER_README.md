# Tangerine Teams — Tester Onboarding

## What you're testing

**Tangerine Teams App v1.9.1** — a desktop app (Windows; macOS/Linux builds coming in v1.9.x). It's the agent-native team OS: a single brain in markdown that all your AI tools (Cursor, Claude Code, Codex, Windsurf, etc.) read and write to. No new AI subscription. Cross-vendor visibility. Designed so a team of 10 stays aligned without anyone manually copying context between tools.

You're testing four design moats:
1. 30-second onboarding (4 cards, no friction)
2. Markdown-as-brain (you can edit it in Notepad)
3. Cross-vendor parser (one memory across all your AI tools)
4. MCP sampling bridge (route LLM calls through your existing AI subscription)

## Why we asked you

You're an outside fresh pair of eyes. No team member has dogfooded this with the v1.9.1 build yet. We need honest first-impression feedback before public release. If it confuses you, it'll confuse everyone — say so.
