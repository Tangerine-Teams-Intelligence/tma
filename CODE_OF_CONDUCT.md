# Code of Conduct

Adapted from [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## TL;DR

Be technical, be direct, be kind. Disagree on the merits, not the person.

## Standards

- Use builder-direct language. Critique code, not people.
- Assume good faith. Read the diff before responding.
- When you're wrong, say so. When others are wrong, show the evidence.
- Keep threads on-topic. Take meta-discussion to GitHub Discussions.

## Not OK

Personal attacks, sustained harassment, discriminatory language, doxxing, or sharing other people's private info without consent. Out-of-scope politics in technical threads.

## Enforcement

Project maintainer (Daizhe Zou) handles reports. Action ranges from a private nudge to permanent bans depending on severity and pattern. Decisions are final.

## Reporting

Email: conduct@tangerineintelligence.ai

Include: link to the thread, what was said, what outcome you'd want. Reports stay confidential. We aim to respond within 7 days.

## Scope

Applies to repo issues, PRs, discussions, and any TMA-branded community space.

---

This is a small open-source project run as a side effort. We'd rather spend the project's energy on shipping than on enforcement. Please make that easy.
